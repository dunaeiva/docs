The `serverless.mcpGateways.admin` role enables managing MCP servers and access to them.

Users with this role can:
* View info on [MCP servers]({{ link-docs-ai }}ai-studio/concepts/mcp-hub/index#servers), as well as create, update, and delete them.
* View info on [access permissions]({{ link-docs }}/iam/concepts/access-control/roles) assigned for MCP servers and modify such permissions.
* Access MCP servers, including external ones, via MCP Hub.

This role includes the `serverless.mcpGateways.editor`, `serverless.mcpGateways.invoker`, and `serverless.mcpGateways.anonymousInvoker` permissions.
