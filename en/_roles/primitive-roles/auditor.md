The `auditor` role grants permissions to read the configuration and metadata for any Yandex Cloud resources without access to the data.

For instance, users with this role can:
* View info on a [resource]({{ link-docs }}/resource-manager/concepts/resources-hierarchy).
* View the resource metadata.
* View the list of resource operations.

`auditor` is the most secure role that does not grant any access to the [service]({{ link-docs }}/overview/concepts/services) data. This role suits the users who need minimum access to Yandex Cloud resources.
